AO3崩铁短信体生成器

一个为 AO3 作者打造的、所见即所得的0基础代码生成器。


本工具本质上是一个代码生成器，帮助0代码基础的小伙伴生成AO3中的作品皮肤（Workskin）和HTML语言代码。但需要注意的是，基于AO3的网站特性，有几项操作本生成器无法为您代劳，还需要一点点自己动手的部分。

1. 作品皮肤与 HTML 是独立的

AO3的作品皮肤和HTML文本写作是两个独立的模块。如果您还不知道什么是作品皮肤，可以参看【官方解读文档（英文）】或者【我的小红书文档（中文）】。 在设定好人物角色后，生成器会生成一个作品皮肤，您需要按照教程自行将这个作品皮肤发布在你的AO3皮肤中，再在发布作品时使用相应皮肤，并在作品编辑框中写入相应的HTML语言。

2. 关于图床与图片链接

AO3允许在作品中插入图片，但需要通过图片链接来实现。这涉及到外部图床。为帮助更多0代码基础的人使用本工具，在图片插入页内嵌了一个免费图床的链接。选择这个图床只是谷歌推荐的结果，并没有利益关系，也不代表生成器作者为图床背书。如果您有其他习惯的图床网站，也是可以使用的，并没有差别。表情和图片插入功能同理。
*如果使用推荐图床，请务必复制上传图片后的“直接链接”

📖 AO3 发布指南

生成 Skin：在生成器中点击 "2. CSS"，复制生成的 CSS 代码。

创建 Skin：在 AO3 点击 Skins -> Create Workskin，粘贴 CSS 并保存。


发布作品：

在 AO3 发布页的 Associations -> Select Work Skin 中选择你刚才创建的 Skin。

在 Work Text 中切换到 HTML 模式，粘贴 HTML 代码。
